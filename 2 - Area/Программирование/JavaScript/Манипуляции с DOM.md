# Манипуляции с DOM

С помощью JavaScript мы можем гибко манипулировать DOM деревом. Например удалять и добавлять элементы.

## Методы манипуляций

Управление содержимом элемента. Значением также можно сделать верстку.

```jsx
div.innerHTML = 'значение';
```

Изменение текстового содержимого элемента. Сюда вписывать верстку уже нельзя.

```jsx
div.textContent = 'значение';
```

Создание элемента. В качестве значения принимает только тег. Созданный элемент может быть только в единичном экземпляре и не может быть вложен сразу в двух элементах.

```jsx
document.createElement('назватие тега');
```

Удаление элемента, к которому обращаемся.

```jsx
div.remove();
```

Добавление элемента внутрь другого элемента

```jsx
div.appendChild('элемент');
```

Создание фрагмента, который будет в себе иметь добавленные элементы, но при добавлении фрагмента в верстку, он исчезает. По факту вложенные в него элементы добавляются без родителя.

```jsx
document.createDocumentFragment();
```

Изменение CSS свойств

```jsx
div.style.свойство = 'значение';
```
